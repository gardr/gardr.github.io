The secure way of adding external content

Adding external content to your site puts site owners in a position where they no longer are in control of wether their site is available or not.

The pasties projects aim to provide a safer way to add external content such as display advertisments to a site.

  • Pasites-js: a simple module for safely embedding images
  • The Validator: tool for making sure rich media content does not break the host site

Pasties-js

Pasties-js is a simple library for taking external content and wrapping it inside an iframe to ensure that the host site always stays up, regardless of how the content behaves.

Check out the pasties-js repo for the code and samples.
